To add a new post, just add a file in the posts folder with the correct naming convention.

Post file format

Post files should follow the rule <LAYOUT><DATE><TITLE>.html.

<LAYOUT> is the name of a layout file under the current theme/layout folder.

<DATE> is the post creation date in the format YYYYMMDD.

<TITLE> is the title of the post.

Limitations

WHile the naming convention is simple and practical, it imposes a technical limitation because some characters are not allowed as file names. I'll probably come up with a syntax expressing such characters so they are substituted when generating the pages.
