Layout files are templates on how to generate posts. For this simple demo site, this is the source code for the template used for the posts. 

<html>
  {{include "include/header.html"}}
  <body>
    <div class="content">
      {{include "include/top.html"}}
      <h2>{{post.title}}</h2>
      <time datetime="{{post.year}}-{{post.month}}-{{post.day}}:00:00+02:00">
        {{post.year}}-{{post.month_name}}-{{post.day}}
      </time>
      <article>{{post.body}}</article>
      <a href="index.html"/>Back</a>
      {{include "include/footer.html"}}
    </div>
  </body>
</html>

It {{include}} the top.html with renders the top banner, and the navigation links. Then it displays the title of the post with the {{post.title}} tag. Then the post creation date with {{post.year}}, {{post.month_name}} and {{post.day}} tags. Then, finally it renders the content of the post with the {{post.body}} tag, right before {{include}}'ing the footer
